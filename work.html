Website built for Greensboro, NC based Trivium Racing to showcase their very fun Glow in the Park night time race series

Bootlegger5k.com

Another website built for Trivium Racing for their 1920s prohibition era rum running Bootlegger 5k trail race.

MeghanMace.com

Site built to show off the work of East Michigan based photographer Meghan Mace.

YouTube Play Icon WordPress Plugin

A WordPress plugin that adds a play icon to the beginning of the page title whenever an embedded YouTube video is playing. When all videos on the page have stopped playing, the play icon is immediately removed from the page title. This mimics the page title behavior when playing a video on youtube.com.

Shingle Visualizer WordPress Plugin

A WordPress plugin for roofing companies. It allows visitors to their site to choose from various house styles, shingle styles and shingle colors, then immediately renders an image of what the house would look like with shingles of the selected style and color on its roof. A try-before-you-buy kind of deal.

Better My Sites Menu WordPress Plugin

Plugin by Kyle Maurer from Real Big Marketing that enhances the My Sites menu that ships with WordPress by automatically shortening the names of websites that are too long. I contributed some custom javascript functions to the project that add an accelerated hover scroll effect to the list of sites so you can always reach the ones at the bottom, regardless of the number of sites you manage, or the size of your browser window.
